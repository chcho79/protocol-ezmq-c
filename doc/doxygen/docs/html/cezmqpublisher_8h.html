Publish an events on list of topics on socket for subscribers.

On any of the topic in list, if it failed to publish event it will return CEZMQ_ERROR/CEZMQ_INVALID_TOPIC.

Parameters
pubHandle- Publisher handle
topicList- List of Topics on which event needs to be published.
listSize- Size of topicList.
event- event to be published.
Returns
CEZMQErrorCode - CEZMQ_OK on success, otherwise appropriate error code.
Note
(1) Topic name should be as path format. For example: home/livingroom/
(2) Topic name can have letters [a-z, A-z], numerics [0-9] and special characters _ - . and /
EZMQ_EXPORT CEZMQErrorCode ezmqSetServerPrivateKey ( ezmqPubHandle_t  pubHandle,
const char *  key 
)

Set the server private/secret key.

Parameters
pubHandle- Publisher handle
key- Server private/Secret key.
Returns
CEZMQErrorCode - CEZMQ_OK on success, otherwise appropriate error code.
Note
(1) Key should be 40-character string encoded in the Z85 encoding format
(2) This API should be called before start() API.
(3) This API should be called, if and only if ezmq is built in secured mode otherwise, it will return CEZMQ_ERROR.
